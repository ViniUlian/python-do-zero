# Python do Zero

Este repositório contém minhas **anotações, exemplos e exercícios práticos** realizados durante o estudo do livro:

**Curso Intesivo de Python: Uma introdução prática e baseada em projetos à programação**
Autor: *Eric Matthes*

Os códigos foram desenvolvidos utilizando o **Google Colab** com o objetivo de registrar meu aprendizado em Python desde o nível iniciante.

---

## Objetivos do Repositório
- Aprender Python desde os fundamentos
- Praticar lógica de programação
- Registrar a evolução do aprendizado
- Criar uma base sólida para projetos futuros

---

## Estrutura do Conteúdo

Os notebooks estão organizados de acordo com os capítulos do livro:
- Variáveis e tipos de dados
- Listas, tuplas e dicionários
- Estruturas condicionais e laços
- Funções
- Classes e orientação à objetos
- Manipulação de arquivos
- Tratamento de erros
- Introdução a projetos práticos

Cada notebook contém:
- Explicação em Markdown
- Exemplos práticos
- Exercícios resolvidos
- Comentários explicando o raciocínio

## Tecnologias Utilizadas
- Python
- Google Colab
- Jupyter Notebook
